# Bomberman DOM

## Project Information
This repository contains a web-based implementation of the classic Bomberman game, allowing players to engage in strategic bomb placement and chaos within a grid-based arena.

## Features
- **Multiplayer Support**: Play with friends locally or online.
- **AI Opponents**: Compete against computer-controlled rivals.
- **Power-ups**: Collect various power-ups to enhance gameplay.
- **Levels**: Multiple levels with increasing difficulty and challenges.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/m7madnat/bomberman-dom.git
   cd bomberman-dom
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## How to Run It
1. Start the application:
   ```bash
   npm start
   ```
2. Open your web browser and navigate to `http://localhost:3000`.

## Project Structure
- `src/`: Contains all source files for the game.
- `public/`: Static files like HTML, CSS, and images.
- `tests/`: Unit and integration tests for the game.

## Tech Stack
- **JavaScript**: The primary programming language for the frontend logic.
- **HTML/CSS**: For structuring and styling the game interface.
- **Node.js**: Used for backend services if applicable.
- **React**: For building reusable UI components.

## Gameplay Instructions
1. **Objective**: Place bombs to defeat opponents and destroy obstacles.
2. Use arrow keys to navigate your character around the grid.
3. Press the space bar to place a bomb.
4. Collect power-ups that appear after destroying blocks.
5. The winner is the last player standing!
